# Homieux Media — Le Rythme des Revenus

Application web simple pour distributeurs d'équipements belges.

## ⚠️ IMPORTANT

Ce fichier `index.html` est **complet et autonome**. Il contient :
- Tout le CSS intégré (pas de fichier externe nécessaire)
- La police Epilogue chargée depuis Google Fonts
- Tous les styles responsive

## Déploiement rapide

1. Copier le contenu de `index.html` dans votre repository
2. **Aucune autre étape** — le CSS est intégré
3. Activer GitHub Pages (Settings → Pages → main branch)
4. Votre site est en ligne
