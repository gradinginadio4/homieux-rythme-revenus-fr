# Homieux Media — Le Rythme des Revenus

Application web simple pour distributeurs d'équipements belges.

## À propos

Cet outil présente une observation courante dans la distribution d'équipements : 
la concentration naturelle de l'activité au deuxième trimestre et les périodes 
plus calmes en fin d'année.

Ce n'est pas un problème à résoudre. C'est un rythme que l'on observe 
régulièrement, et que certaines entreprises accompagnent avec succès.

## Pour qui ?

Entreprises belges de distribution d'équipements (50-500 collaborateurs) 
souhaitant mieux comprendre leur cycle d'activité.
